Project Overview

The Medication Tracking system is a Java based application designed to simulate the management of patients, doctors, medications, and prescriptions
with a pharmacy or medical clinic. In this project we demonstrate Object-Oriented Programming principles such as inheritance, encapsulation,
composition, and modularity.

The system allows users to:

    - Add and manage patients, doctors, medications
    - Create prescriptions linking doctors, patients, and medications
    - Check for expired medications
    - Generate summary reports and perform basic system queries

System Architecture

| Class                      | Description          | Key Responsibilities                                           |
| -------------------------- | -------------------- | -------------------------------------------------------------- |
| `Person`                   | Abstract base class  | Provides shared fields and methods for `Doctor` and `Patient`. |
| `Patient`                  | Subclass of `Person` | Stores patient details, medications, and prescriptions.        |
| `Doctor`                   | Subclass of `Person` | Stores doctor information and their list of patients.          |
| `Medication`               | Independent entity   | Holds medication data, including quantity and expiry date.     |
| `Prescription`             | Composition class    | Links a doctor, patient, and medication together.              |
| `MedicationTrackingSystem` | Core manager class   | Stores collections and manages CRUD operations.                |
| `EmptyMenu`                | Entry point          | Provides a console-based user interface for interaction.       |

User Interface Menu

1           Add new patient
2           Add new doctor
3           Add new medication
4           Generate full system report
5           Check for expired medications
6           Create a new prescription
7           View prescriptions by doctor
8           Restock all medications
9           View prescriptions by patient
10          Exit the system
